# IDE Plugins for WSL (Workflow Specific Language)

This repository contains IDE plugins that provide language support for the Kuetix Workflow Specific Language (WSL).

## Available Plugins

### JetBrains IDEs
* [JetBrains WSL](/jetbrains-wsl) - Plugin for IntelliJ IDEA, GoLand, WebStorm, and other JetBrains IDEs
* [JetBrains WSL README.md](/jetbrains-wsl/README.md)

### Visual Studio Code
* [VS Code WSL](/vs-code-wsl) - Extension for Visual Studio Code
* [VS Code WSL README.md](/vs-code-wsl/README.md)

## Web Playground

### Interactive Browser-Based Editor
* [WSL Playground](/playground) - Web-based playground for trying WSL/SimplifiedWSL without installation
* [Playground README.md](/playground/README.md) - Documentation and embedding guide
* **Features:**
  - 🎨 Real-time syntax highlighting
  - 🌓 Light/dark theme support
  - 🔗 Shareable code URLs
  - 📱 Responsive design
  - 🚀 Embeddable via iframe
  - ⚡ Zero installation required
